## 제품 한 줄 소개

> **"답변만 해주는 챗봇이 아닙니다. 함께 일하는 동료입니다."**

CodePilot은 개발자가 일하는 화면 안에서 **질문 → 구현 → 검증 → 수정**까지 이어지는 전 과정을 함께 수행하는 **AI 기반 업무 파트너**입니다.

## 어떤 문제를 해결하나요?

### 반복적인 작업으로 인한 시간 낭비

- 기능 구현 전 요구사항 정리 및 설계 시간
- 유사 코드 작성, 리팩터링, 테스트 케이스 작성 반복
- 오류 원인 파악과 수정 재시도에 뺏기는 집중 시간

### 팀마다 제각각인 개발 방식

- 팀원별 코드 스타일 차이로 인한 리뷰 비용 증가
- 동일 기능 중복 구현 및 유지보수 난이도 상승
- 보안 정책이나 금지 패턴이 개인 습관에 따라 누락됨

<Tip>
**CodePilot의 해결책**

CodePilot은 **"개인 생산성"**과 **"팀 표준 준수"**를 동시에 달성하여, 개발자는 핵심 로직에만 집중하고 나머지는 AI와 시스템이 처리하도록 돕습니다.
</Tip>

## 조직 단위로 쓰면 달라지는 것

관리자가 Admin 콘솔에서 정책을 설정하면, 조직에 연결된 모든 IDE와 CLI에 자동으로 적용됩니다.

| 관리자가 정하는 것 | 개발자가 얻는 것 |
|-------------------|----------------|
| 사용 가능한 AI 모델 | 검증된 모델만 안전하게 사용 |
| 보안 규칙 | 위험 명령·민감 파일 자동 차단 |
| 코딩 컨벤션 | 팀 표준에 맞는 코드 자동 생성 |
| 사내 문서(RAG) | 우리 회사 맥락을 아는 답변 |

정책은 **필수**와 **권장** 두 가지 강도로 배포됩니다. 필수 항목은 개발자가 끌 수 없고, 권장 항목은 기본값으로 적용되되 필요 시 바꿀 수 있습니다.
