---

조직에 연결하면 관리자가 Admin 콘솔에서 설정한 **AI 모델 정책 · 보안 규칙 · 코딩 컨벤션 · 사내 문서(RAG)** 가 IDE에 자동으로 적용됩니다.

<Info>
개인 사용자는 조직 없이도 CodePilot을 사용할 수 있습니다. 조직 연결은 팀 표준을 적용받기 위한 선택 단계입니다.
</Info>

---

## 1. 조직 API 키 받기

조직 관리자에게 **`cpk_` 로 시작하는 API 키**를 전달받습니다. 관리자는 Admin 콘솔의 **API 키 관리** 메뉴에서 이 키를 발급합니다.

<Warning>
API 키는 조직 가입 자격을 증명하는 값입니다. 사내 메신저·공개 저장소 등에 노출되지 않도록 주의하세요.
</Warning>

---

## 2. IDE에서 조직 가입

<Steps>
  <Step title="설정 열기">
    채팅 패널 오른쪽 위의 **톱니바퀴(⚙) 아이콘**을 클릭해 설정 패널을 엽니다.
  </Step>

  <Step title="계정 탭으로 이동">
    왼쪽 메뉴에서 **계정** 을 선택합니다.

    ![계정 탭](/images/IDE_ACCOUNT.png)
  </Step>

  <Step title="API 키 입력">
    **조직 연결** 항목의 입력창에 전달받은 API 키(`cpk_...`)를 붙여넣고 **등록** 을 클릭합니다.

    {/* SCREENSHOT: 설정 > 계정 탭의 "조직 연결" 영역 — cpk_ 플레이스홀더가 보이는 입력창과 등록 버튼 */}
    ![조직 API 키 등록](/images/IDE_ACCOUNT_ORG.png)
  </Step>

  <Step title="연결 확인">
    가입에 성공하면 프로필 영역에 **조직 이름**이 표시되고, 조직 설정이 자동으로 내려받아집니다.
  </Step>
</Steps>

<Tip>
조직 설정이 바뀌었는데 IDE에 반영되지 않는다면, 계정 탭의 **조직 설정 동기화** 버튼을 눌러 최신 정책을 다시 받아오세요.
</Tip>

---

## 3. 프로젝트 선택

조직에 여러 프로젝트가 있다면, 작업 중인 프로젝트를 선택해야 해당 프로젝트 전용 설정이 적용됩니다.

계정 탭의 **프로젝트** 항목에서 드롭다운을 열어 프로젝트를 선택합니다.

![프로젝트 선택](/images/IDE_PROJECT_SELECT.png)

프로젝트를 선택하면 해당 프로젝트의 AI 모델·보안 규칙·RAG 문서가 적용됩니다. 프로젝트에 별도 설정이 없는 항목은 **팀 기본 설정**을 그대로 따릅니다.

<Info>
드롭다운에 **"등록된 프로젝트 없음"** 이 표시된다면, 관리자가 아직 프로젝트를 만들지 않았거나 회원님이 해당 프로젝트의 멤버가 아닌 경우입니다. 관리자에게 멤버 추가를 요청하세요.
</Info>

## 적용되는 설정

조직에 연결되면 아래 항목이 자동으로 적용됩니다.

| 항목 | 설명 |
|------|------|
| **AI 모델** | 조직이 승인한 모델만 선택 가능 |
| **보안 규칙** | 위험 명령어 차단, 민감 파일 보호 |
| **코딩 컨벤션** | 팀 규칙이 AI 응답에 자동 반영 |
| **사내 문서(RAG)** | AI가 사내 문서를 참고해 답변 |
| **MCP 서버** | 조직이 승인한 외부 도구 연동 |

각 항목은 **필수** 또는 **권장** 강도로 배포됩니다. 필수 항목은 IDE에서 변경할 수 없고, 권장 항목은 기본값으로 적용되지만 필요 시 바꿀 수 있습니다.

---

## 조직 탈퇴

설정 → 계정 → 조직 연결 항목에서 **탈퇴** 를 클릭합니다. 탈퇴하면 조직 정책 적용이 해제되고 개인 설정으로 되돌아갑니다.
