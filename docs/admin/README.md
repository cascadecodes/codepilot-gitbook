# 👮 CodePilot Admin 개요

## 1) 제품 한 줄 소개

> **"우리 회사 AI 개발 환경의 관제탑"**

CodePilot Admin은 조직이 AI 개발 도구를 **안전하고 일관되게 운영**하도록 돕는 통합 관리 콘솔입니다.  
개발자 개개인이 제각각 사용하는 AI를 **"회사 기준에 맞는 팀 자산"**으로 전환합니다.

![Admin 대시보드 예시](https://placehold.co/800x450/EFEFEF/666666?text=Admin+Dashboard)

## 2) 왜 Admin이 필요한가요?

AI 도입이 확산될수록 조직은 새로운 고민에 빠집니다.

| 문제점 | Admin의 해결책 |
| :--- | :--- |
| **품질 편차** | 팀마다 다른 모델/프롬프트를 써서 결과물이 제각각임 | **표준 정책 배포**로 일관성 확보 |
| **보안 공백** | 개인이 알 수 없는 외부 도구를 무분별하게 연결함 | **허용 도구/명령어 중앙 통제** |
| **비용 관리** | 누가 얼마나 쓰는지 몰라 비용 폭탄 우려 | **사용량 모니터링** 및 한도 관리 |

{% hint style="info" %}
CodePilot Admin은 **정책 관리 + 운영 모니터링 + 권한 통제**를 한곳에서 해결하여,  
기업이 안심하고 AI를 전사 도입할 수 있는 기반을 제공합니다.
{% endhint %}
