# IDE 핵심 기능

## 1. 대화형 개발 지원 (Chat & Coding)

개발자가 자연어로 요구사항을 말하면, AI가 의도를 파악하고 코드를 작성합니다.

* **요구사항 분석**: 모호한 요청을 구체화하여 작업 단계 제안
* **코드 생성**: 초안 코드부터 완성형 코드까지 즉시 생성
* **수정 및 개선**: 기존 코드를 분석하여 리팩터링 제안

{% hint style="info" %}
**고객 가치**

* 아이디어를 코드로 옮기는 **초기 진입 장벽 제거**
* 신입/주니어 개발자도 시니어급 가이드를 받으며 작업 가능
{% endhint %}

---

## 2. 안전한 변경 반영 (Diff & Review)

AI가 작성한 코드를 무조건 반영하지 않습니다. 개발자가 눈으로 확인하고 승인해야 적용됩니다.

* **Diff View**: 변경 전/후를 직관적으로 비교
* **승인/거절**: 개발자가 주도권을 가지고 반영 여부 결정

{% hint style="info" %}
**고객 가치**

* 자동화의 편리함과 **코드 안정성**을 동시에 확보
* "AI가 코드를 망칠까 봐 걱정"하는 불안감 해소
{% endhint %}

---

## 3. 자동화 실행 제어 (Automation Level)

조직의 성숙도에 따라 자동화 수준을 조절할 수 있습니다.

* **Level 1 (보수적)**: 모든 단계를 사용자에게 물어보고 실행
* **Level 2 (적극적)**: 검증된 명령어와 도구는 AI가 판단하여 자동 실행

{% hint style="info" %}
**고객 가치**

* PoC 단계에서는 안전하게, 확산 단계에서는 빠르게 운영 가능
* 팀의 **숙련도에 맞춘 유연한 도입** 지원
{% endhint %}

---

## 4. 팀 지식 기반 응답 (RAG)

우리 회사의 문서, 가이드, 레거시 코드를 학습하여 답변합니다.

* **사내 문서 연동**: 위키, API 문서, 규정집 기반 답변
* **맥락 인식**: 프로젝트 히스토리를 이해하고 맞춤형 코드 제안

{% hint style="info" %}
**고객 가치**

* 인터넷에 있는 뻔한 답변이 아닌, **"우리 회사 맞춤형"** 해결책 제공
* 신규 입사자의 온보딩 시간 단축
{% endhint %}

---

## 5. 보안 및 정책 준수 (Security)

관리자가 설정한 보안 가이드라인 안에서만 동작합니다.

* **위험 명령어 차단**: `rm -rf` 등 시스템 파괴 가능 명령어 제한
* **민감 정보 보호**: API Key, 비밀번호 등 중요 정보 필터링

{% hint style="info" %}
**고객 가치**

* **보안 사고 예방** 및 컴플라이언스 준수
* 개발자가 보안 규정을 일일이 외우지 않아도 시스템이 자동 보호
{% endhint %}
