# IDE 핵심 기능

## 1. 대화형 개발 지원 (Chat & Coding)

개발자가 자연어로 요구사항을 말하면, AI가 의도를 파악하고 코드를 작성합니다.

* **요구사항 분석**: 모호한 요청을 구체화하여 작업 단계 제안
* **코드 생성**: 초안 코드부터 완성형 코드까지 즉시 생성
* **수정 및 개선**: 기존 코드를 분석하여 리팩터링 제안

{% hint style="info" %}
**고객 가치**

* 아이디어를 코드로 옮기는 **초기 진입 장벽 제거**
* 신입/주니어 개발자도 시니어급 가이드를 받으며 작업 가능
{% endhint %}

---

## 2. 지능형 모델 라우팅 (Smart Routing)

사용자가 모델을 일일이 고를 필요가 없습니다. 작업의 성격에 따라 IDE가 최적의 모델을 선택합니다.

* **복잡한 코딩**: GPT-4, Claude 3.5 Sonnet 등 고성능 모델 자동 전환
* **단순 채팅**: GPT-3.5, Haiku 등 고속 모델 사용
* **보안 모드**: 사내망 접속 시 로컬 LLM(Ollama)으로 자동 전환

{% hint style="info" %}
**고객 가치**

* **비용 절감**: 불필요하게 비싼 모델을 쓰는 낭비 방지
* **사용성 향상**: 모델 선택 고민 없이 작업에만 집중
{% endhint %}

---

## 3. 안전한 변경 반영 (Diff & Review)

AI가 작성한 코드를 무조건 반영하지 않습니다. 개발자가 눈으로 확인하고 승인해야 적용됩니다.

* **Diff View**: 변경 전/후를 직관적으로 비교
* **승인/거절**: 개발자가 주도권을 가지고 반영 여부 결정

{% hint style="info" %}
**고객 가치**

* 자동화의 편리함과 **코드 안정성**을 동시에 확보
* "AI가 코드를 망칠까 봐 걱정"하는 불안감 해소
{% endhint %}

---

## 4. 자동화 실행 제어 (Automation Level)

조직의 성숙도에 따라 자동화 수준을 조절할 수 있습니다.

* **Level 1 (보수적)**: 모든 단계를 사용자에게 물어보고 실행
* **Level 2 (적극적)**: 검증된 명령어와 도구는 AI가 판단하여 자동 실행

{% hint style="info" %}
**고객 가치**

* PoC 단계에서는 안전하게, 확산 단계에서는 빠르게 운영 가능
* 팀의 **숙련도에 맞춘 유연한 도입** 지원
{% endhint %}

---

## 5. 에러 자동 감지 및 수정 (Hot Load)

개발 중 에러가 발생하면 IDE가 이를 먼저 감지하고 해결책을 제시합니다.

* **빌드 에러 감지**: 컴파일 실패 시 로그를 분석하여 수정 코드 제안
* **런타임 에러 분석**: 실행 중 예외 발생 시 스택 트레이스 분석

{% hint style="info" %}
**고객 가치**

* 에러 원인을 찾기 위해 구글링하는 **시간 낭비 제거**
* 개발 흐름이 끊기지 않고 연속적인 작업 가능
{% endhint %}

---

## 6. 팀 지식 기반 응답 (RAG)

우리 회사의 문서, 가이드, 레거시 코드를 학습하여 답변합니다.

* **사내 문서 연동**: 위키, API 문서, 규정집 기반 답변
* **맥락 인식**: 프로젝트 히스토리를 이해하고 맞춤형 코드 제안

{% hint style="info" %}
**고객 가치**

* 인터넷에 있는 뻔한 답변이 아닌, **"우리 회사 맞춤형"** 해결책 제공
* 신규 입사자의 온보딩 시간 단축
{% endhint %}
