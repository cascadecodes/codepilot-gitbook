# CodePilot 제품 소개서

이 문서는 **CodePilot 제품 소개용 GitBook 문서**입니다.  
기술 구현이나 소스 코드 설명이 아니라, 고객이 궁금해하는 다음 질문에 답하도록 구성했습니다.

- 이 제품이 우리 팀의 어떤 문제를 해결하는가?
- 도입하면 개발 현장이 어떻게 달라지는가?
- 관리자는 무엇을 통제할 수 있고, 사용자에게 어떻게 반영되는가?

## 문서 대상

- 고객사 의사결정자(CTO, 개발실장, 팀장)
- 현업 개발팀 리더 및 실무자
- 운영 정책을 관리하는 관리자

## 한눈에 보는 제품 가치

- **개발 속도 향상**: 반복 작업을 자동화해 구현부터 검증까지 소요 시간을 단축
- **품질 표준화**: 팀 규칙을 공통 정책으로 적용해 결과물 편차를 축소
- **보안 리스크 완화**: 위험 작업을 사전에 제어하고 민감 정보 노출 가능성을 감소
- **운영 투명성 확보**: 사용량/오류/접속 기록 기반으로 관리 지표 확보
- **확장성 있는 도입**: 소규모 팀부터 조직 단위까지 단계적으로 확장 가능

## 문서 구성

- `ide-product-overview.md`: IDE 기능을 사용자 관점에서 상세 설명
- `admin-product-overview.md`: Admin 기능을 운영/관리 관점에서 상세 설명
- `admin-to-ide-settings-flow.md`: Admin에서 설정한 정책이 IDE에 반영되는 방식 설명
