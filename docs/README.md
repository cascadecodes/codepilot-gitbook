# CodePilot IDE 개요

## 1) 제품 한 줄 소개

> **"답변만 해주는 챗봇이 아닙니다. 함께 일하는 동료입니다."**

CodePilot IDE는 개발자가 일하는 화면 안에서 **질문 → 구현 → 검증 → 수정**까지 이어지는 전 과정을 함께 수행하는 **AI 기반 업무 파트너**입니다.

![IDE 메인 화면 예시](assets/IDE_MAIN.png)

## 2) 어떤 문제를 해결하나요?

### 반복적인 작업으로 인한 시간 낭비
* 기능 구현 전 요구사항 정리 및 설계 시간
* 유사 코드 작성, 리팩터링, 테스트 케이스 작성 반복
* 오류 원인 파악과 수정 재시도에 뺏기는 집중 시간

### 팀마다 제각각인 개발 방식
* 팀원별 코드 스타일 차이로 인한 리뷰 비용 증가
* 동일 기능 중복 구현 및 유지보수 난이도 상승
* 보안 정책이나 금지 패턴이 개인 습관에 따라 누락됨

{% hint style="success" %}
**CodePilot의 해결책**

CodePilot IDE는 **"개인 생산성"**과 **"팀 표준 준수"**를 동시에 달성하여,  
개발자는 핵심 로직에만 집중하고 나머지는 AI와 시스템이 처리하도록 돕습니다.
{% endhint %}
