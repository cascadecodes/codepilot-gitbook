## @ 멘션 — 컨텍스트 첨부

입력창에 **`@`** 를 입력하면 AI에게 넘길 컨텍스트를 고르는 메뉴가 나타납니다.

![@ 메뉴](/images/IDE_CHAT_AT.png)

| 항목 | 설명 |
|------|------|
| **Files** | 프로젝트 파일을 선택해 컨텍스트에 포함 |
| **Terminal** | 최근 터미널 히스토리와 출력을 참조 |
| **Diagnostics** | 현재 에러·경고 목록을 참조 |
| **Agents** | 등록된 서브에이전트를 호출 |

```
@src/api/auth.ts 이 파일에서 토큰 만료 처리 로직을 개선해줘.
```

### 서브에이전트 호출

특정 역할을 맡도록 등록한 에이전트는 **`@에이전트이름`** 으로 직접 부를 수 있습니다.

```
@security-reviewer 방금 추가한 인증 코드를 검토해줘.
```

에이전트는 설정 → **Agents** 탭에서 직접 등록하거나, 조직 관리자가 전사 공통으로 배포합니다.

<Tip>
에디터에서 코드를 드래그해 선택한 뒤 채팅을 보내면, 선택 영역이 자동으로 컨텍스트에 첨부됩니다.
</Tip>

## / 슬래시 명령어

입력창에 **`/`** 를 입력하면 명령어 메뉴가 나타납니다.

![슬래시 명령어 메뉴](/images/IDE_CHAT_SLASH.png)

### 기본

| 명령어 | 설명 |
|--------|------|
| `/clear` | 대화 초기화 |
| `/init` | 프로젝트를 분석해 `AGENTS.md` 자동 작성 |
| `/memory` | 장기 기억 추가·삭제 |
| `/compact` | 대화를 압축해 토큰 절약 |

### 개발 워크플로우

| 명령어 | 설명 |
|--------|------|
| `/commit` | 변경 사항 커밋 |
| `/commit-push-pr` | 커밋 → 푸시 → PR 생성까지 한 번에 |
| `/review` | 현재 브랜치 변경분을 AI가 리뷰 |
| `/pr-comments` | PR 리뷰 코멘트를 가져와 반영 |
| `/autofix-pr` | CI 실패 로그를 분석해 자동 수정 |
| `/security-review` | 변경 사항의 보안 취약점 검토 |

### Git

| 명령어 | 설명 |
|--------|------|
| `/git status` | 현재 Git 상태 |
| `/git diff` | 변경 사항 보기 |
| `/git log` | 최근 커밋 히스토리 |
| `/git branch` | 브랜치 목록 |
| `/git stash` | 작업 임시 저장 |

### 세션

| 명령어 | 설명 |
|--------|------|
| `/sessions` | 저장된 세션 목록 |
| `/restore-session` | 이전 세션 복원 |
| `/export` | 대화 내보내기 |
| `/rewind` | 특정 턴 시점으로 되돌리기 |
| `/fork` | 특정 턴에서 대화 분기 |

### 설정 및 도구

| 명령어 | 설명 |
|--------|------|
| `/mcp` | 연결된 MCP 서버 목록·연결·해제 |
| `/agents` | 등록된 에이전트 관리 |
| `/skills` | 스킬·규칙 관리 |
| `/hooks` | 훅 관리 |
| `/cache` | 컨텍스트 캐시 확인·초기화 |

## 입력창 도구

입력창 아래 버튼으로 이번 메시지에만 적용할 옵션을 켤 수 있습니다.

{/* SCREENSHOT: 채팅 입력창 하단 툴바 — 웹검색 토글, Thinking 토글, 이미지/PDF 첨부 버튼이 보이는 상태 */}
![입력창 도구](/images/IDE_INPUT_TOOLBAR.png)

| 도구 | 설명 |
|------|------|
| **웹검색** | 답변 생성 시 웹을 검색해 최신 정보 반영 |

| **이미지/PDF 첨부** | 화면 캡처나 문서를 함께 전달 |

## 에디터에서 바로 쓰기

코드를 드래그해 선택하고 **마우스 오른쪽 버튼**을 누르면 CodePilot 메뉴가 나타납니다.

{/* SCREENSHOT: 에디터에서 코드 블록을 선택하고 우클릭했을 때 나오는 CodePilot 컨텍스트 메뉴 */}
![에디터 컨텍스트 메뉴](/images/IDE_EDITOR_CONTEXT_MENU.png)

| 메뉴 | 설명 |
|------|------|
| **Chat with Selection** | 선택한 코드를 첨부해 채팅 시작 |
| **Explain Code** | 선택한 코드 설명 |
| **Fix Code** | 문제점을 찾아 수정 |
| **Refactor Code** | 리팩터링 제안 |
| **Add Tests** | 테스트 코드 생성 |
| **Add Documentation** | 주석·문서 추가 |

## 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+Enter` (macOS: `Cmd+Enter`) | 모든 변경 적용 |
| `Ctrl+Backspace` (macOS: `Cmd+Backspace`) | 모든 변경 되돌리기 |
| `Tab` | 인라인 코드 자동완성 수락 |
| `Ctrl+Shift+P` (macOS: `Cmd+Shift+P`) | 명령 팔레트에서 CodePilot 명령 실행 |
